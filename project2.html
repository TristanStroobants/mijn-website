Slimme temperatuurmeting

Voor dit project heb ik een slimme temperatuurmeter gemaakt met een ESP32. De ESP32 leest de temperatuur uit via een temperatuursensor en toont de waarde op een eenvoudige webpagina. Tijdens dit project heb ik geleerd hoe ik een sensor aansluit, hoe ik data uitlees met code en hoe ik die informatie zichtbaar maak voor de gebruiker. Dit project past goed bij Internet of Things, omdat hardware, software en internet met elkaar worden verbonden. De temperatuurmeter kan later uitgebreid worden met extra functies, zoals een waarschuwing bij te hoge temperatuur of het opslaan van meetgegevens.
